# Component Cache

Private/reference cache for discovered free/open components.

This folder may be auto-updated by the private sync script.

Expected structure:

```txt
references/component-cache/
└── <source>/
    └── <component-slug>/
        ├── component.json
        ├── README.md
        ├── source.md
        ├── install.txt
        ├── provider-prompt.md
        ├── design-dna.json
        └── code/    # optional; only free/open safe code
```

Do not store paid/pro/private code here.
