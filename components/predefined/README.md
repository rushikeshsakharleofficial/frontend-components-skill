# Predefined Components

Curated reusable component recipes live here.

Each predefined component should be compact and safe to publish:

```txt
<component-slug>/
├── component.json
├── README.md
├── source.md
├── install.txt
├── design-dna.json
└── code/        # optional; only if free/open and publish-safe
```

Avoid large code dumps. Prefer metadata, install command, prompt summary, and adaptation notes.
