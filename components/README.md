# Components Library

This folder stores publishable component knowledge for the skill.

Use this folder for compact, reviewed, reusable component recipes only.

Runtime internet discoveries should first be stored outside the published skill under:

```txt
.aiskill-data/frontend-components/components/discovered/
```

After validation and review, useful components can be promoted into:

```txt
components/predefined/
```

Do not store paid/pro/private components here.
